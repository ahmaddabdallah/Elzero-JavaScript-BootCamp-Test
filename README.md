## 🧭 How to Navigate

### 📚 Lessons Organization

- **Weekly Structure**: Lessons are organized by weeks (Week 01-22)
- **Topic-Based**: Alternative organization in "All Lesson" folder
- **File Naming**: Each lesson has descriptive names with lesson numbers

### 📝 Assignments Organization

- **Range-Based**: Assignments grouped by lesson number ranges
- **Progressive Difficulty**: From basic concepts to advanced projects
- **Mixed Content**: HTML, JS, and JSON files in each assignment set

### 🔍 Finding Specific Content

- **By Topic**: Use "All Lesson" folder for topic-specific content
- **By Week**: Use weekly folders for chronological learning
- **By Assignment**: Use assignment ranges for practice exercises

## 💻 Code Examples

### 🔰 Week 01 - Introduction

```javascript
// 001 - Introduction And What Is JavaScript.js
console.log("Hello from JavaScript!");

// 005 - Where To Put The Code.js
// Internal JavaScript
<script>
    console.log("Internal JavaScript");
</script>

// External JavaScript
<script src="script.js"></script>
```

### 🔶 Week 07 - Arrays

```javascript
// Array Methods and Manipulation
let fruits = ['Apple', 'Banana', 'Orange'];

// Adding elements
fruits.push('Mango');
fruits.unshift('Strawberry');

// Removing elements
fruits.pop();
fruits.shift();

// Array iteration
fruits.forEach((fruit, index) => {
  console.log(`${index + 1}: ${fruit}`);
});
```

### 🔴 Week 21 - APIs & JSON

```javascript
// 177 - Request And Response From Real API.js
let myRequest = new XMLHttpRequest();

myRequest.open('GET', 'https://api.example.com/data');
myRequest.send();

myRequest.onreadystatechange = function () {
  if (this.status === 200 && this.readyState === 4) {
    console.log('JSON Object Content Here');
    console.log(JSON.parse(this.responseText));
  }
};
```

## 🚀 Quick Start

1. **Navigate to specific content**

   ```bash
   # For lessons
   cd lessons/Week01

   # For assignments
   cd Assignments/Assignments\ From\ [\ 001\ ]\ To\ [\ 009\ ]
   ```

2. **Open HTML files in browser**

   ```bash
   # Use a local server to avoid CORS issues
   python -m http.server 8000
   # Then visit: http://localhost:8000
   ```

3. **Follow the learning path**
   - Start with Week 01 lessons
   - Complete corresponding assignments
   - Progress through weeks sequentially
